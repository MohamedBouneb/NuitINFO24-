C'est quoi notre idée ?

Protégez votre site avec un captcha basé sur une idée originale.

Le principe du captcha est simple : nous avons une carte allant de l'Amérique à l'Europe. L'objectif est de traverser l'océan avec la tortue sans toucher les sacs plastiques. Si vous réussissez, vous serez redirigé vers le site Viveris. En cas d'échec, un bouton "Rejouer" apparaîtra pour recommencer le captcha.
